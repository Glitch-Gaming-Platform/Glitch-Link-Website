# Link Website for Your Game

Easily create a website that links all your game's digital assets in one place. This includes store fronts (Steam, Itch, Epic, etc.), social media profiles, and pages where users can sign up for playtests and newsletters.

## About

This repository provides a simple template for a link website. You can host it yourself by forking or downloading the repository and customizing the `index.html` file. Replace the placeholder links with your own, add your images and text, and then upload the website to a hosting service of your choice.

The template is minimal, leveraging [Font Awesome](https://fontawesome.com/) for icons and [Tailwind CSS](https://tailwindcss.com/) for styling.

## How to Use

1. **Fork or Download the Repository**  
   Clone the repository to your local machine or download it as a ZIP file.

2. **Customize the `index.html` File**  
   - Replace the placeholder links with your game's store links, social media profiles, playtest sign-up pages, or any other relevant URLs.
   - Add your own images and update the text content to match your game's branding.

3. **Host the Website**  
   Upload the customized files to a hosting service of your choice. Some popular options include:
   - [GitHub Pages](https://pages.github.com/)
   - [Netlify](https://www.netlify.com/)
   - [Vercel](https://vercel.com/)

## Features

- **Simple and Minimalistic**: Easy to customize and perfect for a straightforward link website.
- **Font Awesome Integration**: Add icons to make your links visually appealing.
- **Tailwind CSS Styling**: Clean, responsive, and modern design.

## Example Use Cases

- **Game Store Links**: Steam, Itch, Epic Games, or any other storefront.
- **Social Media Profiles**: Twitter, Instagram, YouTube, TikTok, etc.
- **Playtest and Newsletter Sign-Ups**: Provide direct links for players to engage with your game.
